# gentoo
some useful stuff for Gentoo Linux, including OpenRC init scripts

# Init scripts
If you're running OpenRC, you can use the following init scripts. To enable one of them, just run `rc-update add {init-script-name} {runlevel}`, e.g. `rc-update add upmpdcli default`.

## upmpdcli
An init script for `upmpdcli`, see https://github.com/cyrevolt/gentoo-overlay
